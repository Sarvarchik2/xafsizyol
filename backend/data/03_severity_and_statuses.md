# Severity Levels and Report Statuses

## Severity Levels

### Small (Kichik / Небольшая)
- Minor crack or small pothole
- Road is still driveable with care
- Low urgency, but should still be fixed to prevent worsening
- Uzbek: Kichik
- Russian: Небольшая

### Medium (O'rta / Средняя)
- Noticeable pothole that requires caution
- Can damage car tires and suspension if hit at speed
- Should be repaired within weeks
- Uzbek: O'rta
- Russian: Средняя

### Critical (Kritik / Критичная)
- Dangerous, deep or wide pothole
- Serious risk of accidents, tire blowouts, suspension damage, and wheel damage
- Urgent repair needed — highest priority for authorities
- Uzbek: Kritik
- Russian: Критичная

## Report Statuses

### Pending (Kutilmoqda / Ожидает рассмотрения)
- Default status when a new report is submitted
- Report has not yet been reviewed by road authorities
- Displayed in yellow/orange color
- The report is visible on the public map

### In Progress (Ko'rib chiqilmoqda / На проверке)
- Authorities have acknowledged the problem and started working on it
- Could mean inspection, planning repairs, or active repair work
- The citizen receives a Telegram notification when status changes to In Progress
- Displayed in blue color

### Fixed (Bajarildi / Исправleno / Fixed)
- The road problem has been resolved and the road is repaired
- The citizen receives a Telegram notification when status changes to Fixed
- Report remains visible on the map but marked as Fixed
- Displayed in green color
- Can still be filtered and viewed in the "Fixed" map filter

## Status Lifecycle

New report → Pending → In Progress → Fixed

Notifications are sent to the citizen at each status change (if they have a Telegram account connected).
